# Weather Component

**Say goodbye to generic forecasts — design your own weather view.**
preview: [weather component](weathercomp.netlify.com)
---

## Overview

This React component allows users to display weather information at your current location with full control over the layout and design. 

---

## Features
- The layout is saved in localStorage for persistence
- Simple, flexible, and easy to integrate into any React app.
- Responsive and clean UI design.
- Built with React and TypeScript for type safety.
